# The Shattered Realm

A portrait-oriented mobile web puzzle game prototype.

## Play

Open `index.html` in a modern browser. On a phone, copy the folder to any static
web host and open its URL.

## Included trials

- **Runes of Allegiance:** sort the four houses onto matching spires.
- **Dragonfire:** aim and launch fireballs to break every winter shield.
- **Realm restoration:** both trials contribute resources and rebuild Blackthorn Keep.

Progress and best scores are saved in the browser.
